Timestamps

A microservice for converting between UNIX and human readable times.

Alternatively, send a GET request to https://vcooley-timestamps.herokuapp.com/time/[TIME]

Replace [TIME] with the time of interest to you. It can be a unix timestamp or a natural date.


The response will contain JSON data with parsed date information as follows:

{"unix": 0, "natural": "January 1, 1970"}
